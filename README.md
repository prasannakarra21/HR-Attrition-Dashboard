# HR Attrition Dashboard
📊 **Project Overview**  
This project showcases an interactive **Power BI dashboard** built to analyze employee attrition trends. Using sample HR data, the report highlights attrition patterns by department, job role, age group, and performance rating.  

## 🔹 Features
- **[DAX measures](ca://s?q=Explain_DAX_measures_in_Power_BI)** such as `CALCULATE`, `REMOVEFILTERS`, and `KEEPFILTERS` for dynamic calculations.  
- **[Interactive visuals](ca://s?q=Interactive_visuals_in_Power_BI)** including bar charts, KPIs, and slicers for drill-down analysis.   
- **[Actionable insights](ca://s?q=Actionable_insights_from_Power_BI_dashboard)** for HR teams to identify high-risk groups and improve retention strategies.  

## 🔹 Repository Structure
- `/pbix` → Power BI report file (`HR_Attrition.pbix`)  
- `/data` → Sample HR dataset (CSV/Excel)  
- `/images` → Dashboard screenshots  
- `/docs` → Documentation and notes
  
- # 🔹 How to Use
1. Download the `.pbix` file from the `/pbix` folder.  
2. Open it in **Power BI Desktop**.  
3. Explore interactive filters and visuals.

## 🔹 Insights
- Attrition is highest among younger employees in certain departments.  
- Performance rating and job role strongly influence attrition risk.  
- HR teams can use this dashboard to design retention strategies.
